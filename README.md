# SEN306: Software Construction

This repository contains my coursework, projects, and notes for SEN306. The focus of this course is on practical software engineering principles, writing clean code, and building reliable software systems.

## 🧠 Core Topics

Throughout this course, the projects will cover:
* Object-Oriented Programming and robust system design.
* Agile software development principles.
* Implementing and managing efficient data structures (e.g., AVL trees, heaps).
* Safe memory management and low-level system interactions.

## 🛠️ Quick Start

Most projects are designed to be run directly from the terminal.
